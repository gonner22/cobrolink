# CobroLink

Generador **gratis** de páginas de cobro en **USDT (ERC20)**. Sin registro, sin custodia de fondos.

**Sitio en vivo:** https://gonner22.github.io/cobrolink/

## Qué hace

- Crea links de pago con monto, título y QR
- Página de cobro lista para WhatsApp, Telegram o email
- Catálogo de micro-servicios pagados en USDT

## Uso local

```bash
python3 -m http.server 8080
# Abre http://localhost:8080
```
