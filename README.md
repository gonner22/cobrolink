# PayLink

Free **USDT (ERC20)** payment page generator. No signup, non-custodial.

## Features

- Create payment links with amount, title, and QR code
- Shareable payment page for chat, email, or social
- Optional services catalog (configure `config.js` first)

## Local setup

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```

## Deploy

1. Copy `config.example.js` to `config.js`
2. Add your public wallet address and contact note
3. Deploy to any static host (GitHub Pages, Netlify, Vercel, etc.)
